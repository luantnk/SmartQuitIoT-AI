# Security Policy

## Supported Versions

Currently, security updates are provided only for the latest release.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| Older   | :x:                |

## Reporting a Vulnerability

We take the security of our software seriously. If you suspect you have found a vulnerability, please **do not** file a public issue.

Instead, please send a report via email to:
**[luantnk2907@gmail.com]**

Please include:
* A description of the vulnerability.
* Steps to reproduce the issue.
* (Optional) Any specific attack vectors or proof-of-concept code.

I will acknowledge your email within 48 hours and will prioritize fixing the issue.
